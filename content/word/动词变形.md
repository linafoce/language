# 日语动词变形总表

> 以三类代表动词为例：一类 `書く`、二类 `食べる`、三类 `する / 来る`

| 变形名称 | 日语称呼 | 主要功能/意义 | 一类（書く） | 二类（食べる） | 三类（する / 来る） |
| :-- | :-- | :-- | :-- | :-- | :-- |
| 辞书形（原形） | 辞書形 | 动词的基本形式，用于查字典或非过去式肯定 | 書く | 食べる | する / 来る |
| `Masu` 形（礼貌形） | ます形 | 礼貌体肯定（敬语） | 書きます | 食べます | します / 来ます |
| `Nai` 形（否定形） | ない形 | 简体否定（不做……） | 書かない | 食べない | しない / 来ない（こない） |
| `Te` 形（连接形） | て形 | 连接句子、正在进行、请求等 | 書いて | 食べて | して / 来て |
| `Ta` 形（过去形） | た形 | 简体过去式（做了……） | 書いた | 食べた | した / 来た |
| 意志形 | 意向形 | 提议、劝诱、自言自语（吧/想要……） | 書こう | 食べよう | しよう / 来よう（こよう） |
| 命令形 | 命令形 | 强硬的命令（去做……） | 書け | 食べろ | しろ / 来い（こい） |
| 禁止形 | 禁止形 | 强硬的禁止（不许做……） | 書くな | 食べるな | するな / 来るな |
| 假定形（`Ba` 形） | ば形 | 假设条件（如果……的话） | 書けば | 食べれば | すれば / 来れば（くれば） |
| 可能形 | 可能形 | 能力或可能性（能做/会做……） | 書ける | 食べられる | できる / 来られる（こられる） |
| 被动形 | 受身形 | 被动语态（被……） | 書かれる | 食べられる | される / 来られる（こられる） |
| 使役形 | 使役形 | 让某人做某事（让/叫……） | 書かせる | 食べさせる | させる / 来させる（こさせる） |
| 使役被动形 | 使役受身 | 被迫做某事（不得不/被强迫……） | 書かされる | 食べさせられる | させられる / 来させられる（こさせられる） |

## 补充说明

* `来る` 的读音会随变形变化，最常见的是：
  * 来ない（**こ**ない）
  * 来よう（**こ**よう）
  * 来い（**こ**い）
  * 来れば（**く**れば）
  * 来られる（**こ**られる）
  * 来させる（**こ**させる）
* `可能形` 和 `被动形` 里，二类动词都是 `～られる`，要结合语境判断意思。
* `する` 的可能形是特殊形式 `できる`，不能直接按普通规则变。
