### 尊他语 (尊敬語) 学习笔记

#### 1. 核心定义

> **核心逻辑：** 通过 **抬高对方** 的动作、状态或所属物，来直接表达对对方的敬意。

*   **适用对象：** 用于**别人**（长辈、上司、客户等）发出的动作或状态。
*   **注意：** 绝对**不能**用于自己或己方人员。

#### 2. 名词的尊他形式

有些名词本身就带有敬意，或者通过添加前后缀变成敬语。

**① 本身带有敬意的名词/代词**

*   **人称/指代：**

    *   この / その / あの / どの **方（かた）** *(对应：人)*
    *   **どなた** *(对应：だれ)*
    *   こ / そ / あ / **どちら** *(对应：こっち/そっち/あっち/どっち)*
*   **复数敬称：**

    *   先生 **方（がた）** *(对应：たち)*

**② 加前后缀变敬语 (お / ご / ～さん / ～様)**

*   **常见搭配：** お母さん、お父様、田**中様**。
*   **关于慰问：** ご苦労さん/様、お疲れ様。

③ 「お」与「ご」的接头词规则 (重点)

| 接头词   | 接续词性          | 规律 (一般情况) | 典型例子                              |
| :---- | :------------ | :-------- | :-------------------------------- |
| **お** | **训读词** (和语词) | 日本原本固有的词汇 | **お**宅 (宅)、**お**休み (休)、**お**話 (話) |
| **ご** | **音读词** (汉修词) | 源自中文发音的词汇 | **ご**両親、**ご**家族、**ご**案内           |

#### 3. 形容词的尊他形式

用来形容对方的状态、性质或特征。

*   **基本公式：** `前缀 お + A1 (一类形容词/い形) / A2 (二类形容词/な形)`
*   **常见例子：**

    *   **お**忙しい (您很忙)
    *   **お**美しい (您很美)
    *   **お**詳しい (您很懂行/详细)
    *   **お**若い (您很年轻)
    *   **お**強い (您很强)
*   **特殊情况：** 极少数形容词（通常是汉源词性质的）接前缀 **「ご」**。

    *   **例：** **ご**立派 (您很出色/气派)

### 尊他语 (尊敬語) 进阶笔记：动词篇

#### 一、尊他语的三种表现形式（按敬意级别排序）

在将普通动词变为尊他语时，有三种方式，敬意程度依次递减：

| **形式**         | **敬意程度** | **说明**                             | **例句**                              |
| :------------- | :------- | :--------------------------------- | :---------------------------------- |
| **特殊尊他动词**    | 最高级      | 有专门尊他词时优先使用，如 `行く` → `いらっしゃる`。 | 社長はもう**いらっしゃいました**。              |
| **尊他语固定句型**   | 高级       | 即 `お/ご～になる` 等句型。                 | お父さんは何時ごろ**お帰りになりますか**。         |
| **动词被动态**     | 基础级      | 变成“被动形式”，但不翻译成“被……”。             | 先生はもう**帰られました**か。（老师已经回去了吗？） |

#### 二、核心固定句型 A：无实意尊他

| **句型**          | **接续规则（公式）**                 | **特点与注意点**                               | **典型例句**                         |
| :-------------- | :---------------------------- | :--------------------------------------- | :------------------------------- |
| **① お/ご ～ になる** | **お** + V1/V2词干；**ご** + V3词干 | 最常用的标准尊他句型，敬意程度高于 `れる・られる`。        | お客様はもう**お帰りになりました**。          |
| **② お/ご ～ なさる** | 同上                            | 语气较陈旧/郑重。                               | 先生は今、何について**ご研究なさいますか**。     |
| **③ お/ご ～ です**  | 同上                            | 常用于正在进行、反复发生或将来的动作/状态，能搭配的动词比较有限。 | 先生が研究室で**お待ちです**。             |

**`お/ご～です` 常见搭配：** `呼ぶ`、`過ごす`、`待つ`、`帰る`、`読む`、`考える`、`勤める`、`欠席する`、`出勤する` 等。

*   もう**お帰りですか**。（您要回去了吗？）
*   社長も今日の営業会議に**ご出席になります**。（社长也会出席今天的营业会议。）

#### 三、核心固定句型 B：请求与授受

*(注：这几个句型有实意，涉及“请”或“给”)*

**1. 请您做某事： `お/ご ～ ください`**

*   **公式：** `お + V1/V2词干 + ください` / `ご + V3词干 + ください`
*   **对比：** 意思等同于 `～てください`，但前者敬意更高。
*   **⚠️ 绝对禁忌 (考试陷阱)：** **不可混用！**

    *   ✅ `連絡してください` (普通敬体)
    *   ✅ `ご連絡ください` (尊他句型)
    *   ❌ `ご連絡してください` (**错误！** `ご` 和 `して` 重复，画蛇添足)
*   **辨析补充：**

    *   `～てください` = 请**您**做...
    *   `(さ)せてください` = 请让**我**做... (例：少し考えさせてください。)

**2. 别人为我做某事： `お/ご ～ くださる`**

*   **公式：** `お + V1/V2词干 + くださる` / `ご + V3词干 + くださる`
*   **原形来源：** 它是授受动词 `くれる` (给我) 的尊他语。
*   **变形注意：** `くださる` 的ます形是特例：`くださいます`。
*   **例句：** 先生は日本語を**お教えくださいました**。（老师教了我日语。）

## 自谦语 (謙譲語) 学习笔记

#### 1. 核心定义

> **核心逻辑：** 通过 **降低自己**（或己方人员）的动作、状态，来间接抬高对方，表示对对方的敬意。

*   **适用对象：** 用于 **自己** 或 **己方人员**（家人、公司同事等）发出的动作。
*   **记忆口诀：** “我做动作，我自谦”。

#### 2. 名词的自谦形式

*   **自称：** わたくし（私）、者（もの）。
*   **己方单位：** 弊社（へいしゃ - 敝司）。
*   **己方家人：** 父、母、うち（对外人提到自己家人时，不用お父さん等尊称）。
*   **复数后缀：** ～ども（私ども）、～ら（僕ら）。

***

#### 3. 核心固定句型 (高频考点)

这里有四个关键句型，请务必分清\*\*“动作的主语”\*\*是谁。

**① 基础自谦：我做某事 (无实意辅助动词)**

*   **公式：**

    *   `お + V1/V2词干 + する` （或 `いたす` / `申す` / `申し上げる`）
    *   `ご + V3词干 + する` （或 `いたす` ...）
*   **替换词：** `いたす` 是 `する` 的自谦；`申す/申し上げる` 是 `言う` 的自谦，语感更郑重。
*   **例句：**

    *   学校を**ご案内します**。（我带您参观学校。）
    *   よろしく**お願いいたします**。（请多关照。）
    *   **お詫び申し上げます**。（我向您致歉。）

**② 请求句式：希望您做某事 (願う)**

*   **公式：** `お/ご + 词干 + 願う`
*   **常用语气：** `～願えませんか` / `～願えないでしょうか` (能请您...吗？)
*   **例句：**

    *   喫煙は**ご遠慮願います**。（请您勿吸烟 / 希望您忍耐一下吸烟这件事。）
    *   **お伝え願えませんか**。（能请您转达一下吗？）

**③ 领受恩惠：承蒙您做某事 (いただく)**

*   **核心：** `いただく` 是 `もらう` (得到) 的自谦语。
*   **主语：** 句子主语是“我” (我领受了您的动作)。
*   **公式：** `お/ご + 词干 + いただく`
*   **例句：**

    *   ご結婚式に**お招きいただき**...（承蒙您邀请我参加婚礼... / 我领受了您的邀请）
    *   **ご注意いただき**たいこと。（希望您注意的事项 / 我希望领受您的注意）

**④ 允许/使役：请让我做某事 (重要!!)** 这是最“日式”的表达，表示\*\*“在得到您的许可下，让我做...”\*\*。

*   **核心结构：** `动词使役态 + ていただく` (让我做... + 我得到这个恩惠) = **请允许我做**
*   **常见形式：**

    *   `～(さ)せていただきます` (宣言：那我就不客气地做了 / 请允许我做)

        *   例：自己紹介**させていただきます**。（请允许我自我介绍。）
    *   `～(さ)せていただけないでしょうか` (请求：能让我做...吗？) —— **最谦虚礼貌的请求**

        *   例：先に**帰らせていただけないでしょうか**。（能允许我先回去吗？）

***

### 💡 易混淆点辨析 (避坑)

**1. 「くださる」 vs 「いただく」** 这两个词意思一样（别人给我做），但**主语**不同，考试必考！

*   **くださる (尊他)：** 主语是**对方**。\[例] 先生は教えてくださいました。（老师教了我。）
*   **いただく (自谦)：** 主语是**我**。\[例] 私は先生に教えていただきました。（我领受了老师的教导。）

    *   *注：`お/ご～いただく` 比 `～ていただく` 语气更谦逊。*

**2. 两个“请”的区别**

*   **～てください：** 简单的“请您做”。(有时会显得有点命令口吻)
*   **～(さ)せてください：** “请**让**我做”。(请求许可)

    *   *升级版：* `～(さ)せていただけませんか` (能让我做吗？)

### 一、 郑重语 (丁寧語・ていねいご)

这也是我们常说的 `です / ます` 体，用于体现说话人的教养和礼貌。

#### 1. 核心定义

*   通过压低动作主体，直接对**听话人**表示敬意。
*   **特点：** 不管主语是谁（哪怕是描述天气、花草），只要想对听话人礼貌，就可以用。

#### 2. 动词变化

*   **基本体：** `です`、`ます`
*   **高级郑重形式 (特殊变化)：**

    *   **ある** (有/在) → **ございます**
    *   **です / だ** (是) → **でございます**

        *   *过程分解：* `です/ある` → `ござる/でござる` → `ございます/でございます`

> **📝 知识点：语体鄙视链**
>
> *   `だ` (简体)：用于朋友、后辈、小说。
> *   `です` (敬体)：最常用，通用礼貌。
> *   `である` (书面体)：用于正式演讲、论文（接续与 `だ` 相同）。
> *   **尊敬程度：** `だ` < `です` < `である` (注：此处图示意思可能是正式程度，口语中 `でございます` 最尊敬)

***

### 二、 敬语做题技巧 (解题万能公式)

考试或实战时，按这个步骤思考，准确率翻倍：

1.  **看主语 (谁做的动作?)**

    *   **别人**做的 → 用 **尊他语** (抬高)
    *   **我/己方**做的 → 用 **自谦语** (压低)
2.  **选形式 (还原句子)**

    *   是考“固定句型”？还是考“特殊单词”？
    *   **技巧：** 看到 `(ら)れる` 结尾，先反应是 **被动态** → 联想到 **被动表尊他**。
3.  **避坑检查**

    *   **❌ 错误用法：** `お/ご + ～て + ください / いただく / くださる`

        *   (例如：不能说 `ご参加してください`，要说 `ご参加ください`)

***

### 三、 使用敬语的注意事项 (❌ 常见雷区)

这部分是**实战和改错题**的重灾区，请打星号★！

#### 1. "内外有别" 原则 (最重要!)

*   **规则：** 对**外人**谈论自己家人或公司领导时，**不能用尊他语**，要用自谦语或郑重语。
*   **例：**

    *   ❌ 社長は外出していらっしゃいます。(社长出去了 - 对客户说) → 用了尊他，错误！把社长抬高了，等于没把客户放在眼里。
    *   ✅ 社長は外出して**おります**。(社长出去了) → 用自谦 `おる`，正确。

#### 2. 物/动物不仅能用尊他

*   **规则：** 主语是物体、植物、动物时，即便归长辈所有，也**不用尊他语**。
*   **例：**

    *   ❌ 社長のかばんはあそこに**いらっしゃいます**。(社长的包在那里) → 错误，包不是人。
    *   ✅ 社長のかばんはあそこに**ございます**。

#### 3. 复合动词 "敬后不敬前"

*   **规则：** 当动词是 `V + て + 补助动词` 时，通常只把**后面的补助动词**变成敬语。
*   **例：**

    *   ✅ 読んで**いらっしゃいます** (正在读 - `いる` 变尊他)
    *   ❌ **読まれて**います (前变后不变，通常不自然)

#### 4. 禁止 "双重敬语"

*   **规则：** 同一个动词不能套用两层尊他语法。
*   **例：**

    *   ✅ お求めになりました (套用 `お～になる` 句型)
    *   ❌ お求めに**なられ**ました (套用了 `お～になる` + `られる` 被动尊他，重复了！)

#### 5. "特殊词" 优先于 "句型"

*   **规则：** 如果这个词有专门的敬语动词（如 `行く` → `いらっしゃる`），优先用专门的，不要硬套 `お～になる`。
*   **例：**

    *   ✅ 社長は何と**おっしゃいました**か。(社长说什么了？)
    *   ❌ 社長は何とお言いになりましたか。(语法没错，但不够地道/高级)
    *   **★ 例外 (可以套句型的特殊词)：**

        *   `来る` → `お見えになる` (常用)
        *   `越す` → `お越しになる`
        *   `知る` → `ご存知です` (注意不是 `ご存知になる`)

#### 6. 自谦语必须有 "接受者"

*   **规则：** 自谦语是为了向“动作接受者”表示敬意。如果你的动作**与他人无关**（单纯的自我行为），不能用自谦语，要用郑重语。
*   **例：**

    *   (自己在房间里) "我明天去东京。"
    *   ❌ 明日、東京へ**参ります**。(参る是自谦，暗示去见某人或去某地朝拜)
    *   ✅ 明日、東京へ**行きます**。(郑重语)
