# 语法笔记：表推测样态（`～そうだ` / `～ようだ` / `～みたいだ` / `～らしい`）

> 核心意思：`“好像……”`、`“像……一样”`
> 记忆提示：`看前又看后`

---

## 1. `～そうだ`（样态）

### 核心语法

* **动词：** 去「ます」形
* **形1：** 去「い」
* **形2：** 词干
* **名词：** 无名词接续
* **后缀：** `＋ そうだ`
* **活用：**
  * `そうだ` + 名词 = `そうな + 名词`
  * `そうだ` + 动词 / 形容词 = `そうに + 动词 / 形容词`

---

### 用法重点

#### 1. 推测样态

主要用于描述说话者观察到、感觉到的某种情形、样子、迹象、趋势等，即**视觉印象**。

> **注意：** 不和过去式搭配使用，一般不用于单纯回顾已经发生完的事情。

* **例句：**
  > **このお弁当はおいしそうです。**
  > 这便当看起来很好吃。

  > **もうすぐ雨が降りそうです。**
  > 看起来马上要下雨了。

#### 2. 活用变化

* `いい` = **よさそうです**
  > **李さんの頭がよさそうです。**

* `ない` = **なさそうです**
  > **あの人はお金がなさそうです。**

#### 3. 否定形式

**（1）形容词：前否 / 后否**

* **前否：**
  * 形1：`～くなさそうだ`
  * 形2：`～ではなさそうだ`
* **后否：**
  * 形1 去「い」 / 形2 词干 + `そうではない`

**（2）动词：后否**

* 动词去「ます」 + `そうにない`
* 动词去「ます」 + `そうもない`
* 动词去「ます」 + `そうにもない`

> 「に」「も」有时可以单独省略，但考试里常见的是完整形式。

* **例句：**
  > **この小説はおもしろそうではありません。**

  > **李さんは来そうにもありません。**

#### 4. 固定句型

* **`～そうになった / ～そうだった`**：险些…… / 差点儿……
  > **びっくりして、持っていたガラスを落としそうになった。**

* **`～そうに見える`**：看上去好像……
  > **誕生パーティーで彼女は幸せそうに見える。**

---

## 2. `～ようだ`

### 核心语法

* **动词：** 简体
* **形1：** 直接接
* **形2：** `＋ な`
* **名词：** `＋ の`
* **后缀：** `＋ ようだ`
* **活用：**
  * `ようだ` + 名词 = `ような + 名词`
  * `ようだ` + 动词 / 形容词 = `ように + 动词 / 形容词`

---

### 用法重点

#### 1. 表比喻

常和「**まるで**」「**全く**」「**あたかも**」等副词搭配使用。

* **例句：**
  > **宝くじに当たったなんて、あたかも夢のようです。**

  > **彼女の笑顔は太陽のように明るく輝いています。**

* **固定句型：** `まるで～かのようだ`
  > **歓迎会には着物を着た女性がたくさん集まり、会場はまるで花が咲いたかのようでした。**

#### 2. 表列举

列举相同性质、内容示例中的一个或若干个。

* **例句：**
  > **みんなは子供のように元気に歌い始めます。**

  > **彼のような優秀な人材は珍しいです。**

#### 3. 表主观推测

常和「**どうも**」「**どうやら**」等副词呼应使用。

* **例句：**
  > **先週は図書館が休みだったようです。**

  > **外は寒いようですね。**

* **固定搭配：** `～ように見える`
  > **便利なように見えたので、買ってみたが、使ってみるとたいしたことはなかった。**

---

### 易混点：和「ように」长得很像的语法

#### 1. 表目的：为了……

* **接续：** 动词可能态 / 动词原形 / 动词ない形 + `ように`

#### 2. 由“目的”延伸出来的固定句型

* **`～ようになる`**：表示变化、结果、状态（偏客观）
* **`～ようになっている`**：表示功能、性能；或某种状态一直如此
* **`～ようにする`**：努力做到……
* **`～ようにしている`**：已经形成习惯，一直努力做到……
* **`～ようにしてください`**：请做某事

> 有时会省略后面的部分，看起来只剩下一个 `よう`，但本质上还是这一组句型。

#### 3. 表意志的「よう」

也就是动词意志形里的 `よう`。

* **例句：**
  > **これからはもっと日本語を勉強しよう。**

#### 4. 表愿望、期许

有时也会省略 `に`，看起来像单独的 `よう`。

* **例句：**
  > **お母さんの病気が治るように。**

#### 5. 表间接引用、指示等内容

后面常跟「言う」「頼む」「命令する」「注意する」等表达，可理解为接近 `という` 的内容。

* **例句：**
  > **お医者さんにタバコを吸わないように言われます。**

---

## 3. `みたいだ`

### 核心语法

* **动词 / 形1：** 简体
* **名词 / 形2：** 词干
* **后缀：** `＋ みたいだ`
* **活用：**
  * `みたいだ` + 名词 = `みたいな + 名词`
  * `みたいだ` + 动词 / 形容词 = `みたいに + 动词 / 形容词`

---

### 用法重点

#### 1. 表比喻

经常和「まるで」等副词搭配使用。

* **例句：**
  > **まるで夢を見ているみたいです。**

  > **彼女は天使みたいな顔をしています。**

#### 2. 表列举

* **例句：**
  > **妹は体操の選手みたいに体が柔らかいです。**

  > **彼みたいな人が好きです。**

#### 3. 表主观推测

常和「どうも」「どうやら」等副词搭配使用。

* **例句：**
  > **どうやら電車が遅れているみたいです。**

  > **田中さんは料理が上手みたいです。**

> **注意：** `みたいだ` 和 `ようだ` 的基本用法很接近。
> `みたいだ` 更偏**口语**，`ようだ` 更偏**书面语**。

---

## 4. `～らしい`

### 用法重点

#### 1. 表客观推测

依据外来的消息等进行客观推测，常和「どうも」「どうやら」搭配使用。

* **接续：** 动词 / 形1 简体、名词 / 形2 词干 + `らしいです`
* **例句：**
  > **電気が消えているから、母は寝たらしいです。**

  > **その話は、どうも本当らしいです。**

#### 2. 表典型性

接在名词后，构成复合形容词。表示该名词所表示事物的典型特征。

意思接近：像……一样的…… / 真正意义上的…… / 名副其实的……

* **接续：**
  * **不同名词：** 名词1 + `らしい` + 名词2
    > **このあたりはいかにも京都らしい雰囲気です。**

    > **今日は春らしい天気です。**

  * **相同名词：** 名词1 + `らしい` + （名词1）
    > **あの人は本当に作家らしい作家です。**

> **注意：** 后项名词经常被省略。
> 例如：`男らしい男`

---

## 5. 表传闻：`～そうだ` / `～らしい`

> 核心意思：听说…… / 据说……
> 常见信息来源：`～によると`、`～によれば`、`～話では`、`新聞では`

### 1. `～そうだ`（传闻）

* **接续：** 小句简体 + `そうだ`
* **例句：**
  > **吉田さんの彼女はきれいだそうです。**

  > **天気予報によると、明日は晴れるそうです。**

* **注意：** 表示传闻的 `そうだ` 本身没有否定、过去、疑问形式。
  * 要表达“听说不…… / 没……”时，用 `～ないそうだ`

* **例句：**
  > **劉さんは高くないそうです。**

  > **吉田さんは日本に帰ったそうです。**

### 2. `～らしい`（传闻）

* **接续：** 动词 / 形1 简体、名词 / 形2 词干 + `らしい`
* **注意：** `～らしい` 更强调二手消息，语气比 `そうだ` 更委婉一些。
* **例句：**
  > **天気予報によると、明日は雨らしいです。**

  > **医者の話では、薬を飲んでも治らないらしいです。**

* **否定形式：** `～ないらしい`
  > **仕事はまだ終わらないらしいです。**

---

## 6. 推测样态语法总结

### A. 样态（Inference / Appearance）

| 样态 | **そうだ** | **ようだ** | **みたいだ** | **らしい** |
| :-- | :-- | :-- | :-- | :-- |
| **1. 用法** | 单纯表示推测样态，强调“看起来……” | ① 主观推测 ② 列举 ③ 比喻，强调“样子” | ① 主观推测 ② 列举 ③ 比喻 | ① 客观推测 ② 典型性，强调“听来的判断” |
| **2. 接续** | **V 去ます**、A1 去い、A2 / N 词干 | **看成名词**：V 简体、N + **の**、A1、A2 + **な** | **V、A1 的简体**；**N、A2 的词干** | **V、A1 的简体**；**N、A2 的词干** |
| **3. 活用形式** | **そうな** + N；**そうに** + V / A | **ような** + N；**ように** + V / A | **みたいな** + N；**みたいに** + V / A | **らしい** + N；**らしく** + V / A |
| **4. 否定形式** | ① **よさそう / なさそう** ② 形容词可前否 / 后否，动词多用后否 | - | - | - |
| **5. 中顿** | **そうで** | **ようで** | **みたいで** | **らしく** |
| **6. 固定搭配** | **～そうになった / ～そうだった**；**そうに見える** | **まるで～かのようだ**；**ように見える** | **まるで～かみたいだ** | - |

### B. 传闻（Hearsay）

| 传闻 | **そうだ** | **らしい** |
| :-- | :-- | :-- |
| **1. 接续** | **小句简体 + そうだ** | **V、A1 的简体**；**N、A2 的词干** |
| **2. 可靠性** | 更接近直接转述 | 更偏二手消息、委婉判断 |
| **3. 信息来源** | **～によると、～によれば、～話では、新聞では** | 同样可搭配这些来源表达 |
