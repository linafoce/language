# 使被能专题

## 一、使被能专题包含

1.  **使役态**：表示“让、令、使”某人做某事。
2.  **被动态**：表示“被、受”某动作的影响。
3.  **使役被动态**：表示“被迫、被迫做、被要求”做某事。
4.  **可能态**：表示“能、能够、会”做某事。

## 二、基本大句型

**主语（は・が） + 对象（に） + V（动词变形）**

***

## 三、变形规则汇总

| 类别          | V1 (五段)                         | V2 (一段)      | V3 (サ变/カ变)                                            |
| :---------- | :------------------------------ | :----------- | :---------------------------------------------------- |
| **① 使役态**   | 「う」段 `$\rightarrow$` 「あ」段 + せる  | 去「る」 + させる   | する `$\rightarrow$` させる<br>くる `$\rightarrow$` こさせる     |
| **② 被动态**   | 「う」段 `$\rightarrow$` 「あ」段 + れる  | 去「る」 + られる   | する `$\rightarrow$` される<br>くる `$\rightarrow$` こられる     |
| **③ 使役被动态** | 「う」段 `$\rightarrow$` 「あ」段 + される | 去「る」 + させられる | する `$\rightarrow$` させられる<br>くる `$\rightarrow$` こさせられる |
| **④ 可能态**   | 「う」段 `$\rightarrow$` 「え」段 + る   | 去「る」 + られる   | する `$\rightarrow$` できる<br>くる `$\rightarrow$` こられる     |

> **注意：**
>
> *   V1 变形凡是涉及变为「あ」段时，若词尾为「う」，需变为「わ」（如：買う `$\rightarrow$` 買わせる）。
> *   V1 使役被动态有简略形式：あ段 + される（但「す」结尾的动词除外，必须用「させられる」，如：話させられる）。
> *   红色标注：①被动 V2、来ると可能 V2 一样；②被动 V3 すわ和使役被动 V1 一样；③可能 V3 する`$\rightarrow$`できる

## 四、详细用法与详解

### 1. 使役态 (Causative)

*   **分类：** 自动词使役句、他动词使役句。
*   **自动词使役句：**
    *   句型：主语（は・が） + 对象（**を**） + V自（さ）せる
    *   例：弟はよく両親**を**怒らせました。（弟弟经常让父母生气。）
*   **他动词使役句：**
    *   句型：主语（は・が） + 对象（**に**） + V他（さ）せる
    *   例：先生は学生**に**本を読ませます。（老师让学生读书。）
*   **闭环故事（判断助词技巧）：**
    1.  使役态要注意 `$\rightarrow$` 对象的助词。
    2.  对象的助词由“自、他动词”影响。
    3.  自动词用“を”（人为使动）；他动词用“に”。
    4.  “情感类单词、移动动词”多为自动词，对象的助词用“を”。
*   **相关句型：**
    *   `V(さ)せてください`：“请让我/允许我做某事”
    *   `V(さ)せてくださいませんか`：“能请您让我/允许我做某事吗？”（更尊敬的说法）
    *   例：気分が悪いですから、早退させてくださいませんか。

### 2. 被动态 (Passive)

*   **分类：** ①直接被动句、②间接被动句、③物主被动句。

**① 直接被动句：注意“对象” `$\rightarrow$` 三种情况**

*   例：その子は 祖父に 育てられました。
*   例：この近くに マンションが 建てられました。（对象不明确/不需要表达时，省略“に”）
*   例：『阿Q正伝』は魯迅によって 書かれました。（发明创造用“によって”）

**② 间接被动句：注意“翻译”**

*   含义：间接的被一件事影响了，从而带来不好的情感。
*   例：李さんは 父に 死なれました。（小李遭到了父亲去世的打击/父亲去世了，小李很难过。）
*   例：電車の中で、隣の人にタバコを吸われて、気持ちが悪くなりました。

**③ 物主被动句：注意“句型”，且成分单独成立**

*   句型：**私 が 犬 に 手 を 噛まれました。**（主语是人，身体部位用を）
*   练习：私\_\_\_\_手を犬に噛まれました。（此处助词通常省略或为は/が，视语境）

**补充用法：**

*   **被动表尊敬：** 不翻译“被”，只翻译单词本身意思，形式上进行尊敬。
    *   例：先生はいつもどんな新聞を読まれますか。
*   **被动表自发：** 表示某种情感自然而然发生，“不由得...；禁不住...”。
    *   例：小さい頃のことが思い出される。

### 3. 使役被动态 (Causative-Passive)

*   **含义：** “被迫、被逼着、被要求”
*   **用法一：表被迫**
    *   野菜が嫌いな子供は 每日 母に 野菜を 食べさせられます。（讨厌蔬菜的孩子每天被迫吃蔬菜。）
    *   昨日、私は病院で２時間ぐらい 待たされました。
*   **用法二：表自发（情感）**
    *   子供が書いた作文は素晴らしくて、感心させられました。（被迫感到佩服 `$\rightarrow$` 不由得佩服）
*   **补充知识：被动表自发 VS 使役被动表自发**
    1.  小さい頃のこと **が** 思い出される。（事情被我想起）
    2.  小さい頃のこと **を** 思い出させられる。（我被让他想起 `$\rightarrow$` 我想起事情）

### 4. 可能态 (Potential)

*   **变形注意：** 动词的可能态按照二类动词规则进行变形，所以又叫“可能动词”。
*   **助词变化：** 可能态的助词用 **「が」** 提示。
*   **特殊动词：** 存在动词「ある・いる」本身带有“能”含义；「見える・聞こえる・分かる・できる」没有可能态。
*   **句型对比：**
    *   私は日本語 **が** 話せます。
    *   私は日本語 **を** 話すことができます。
*   **关于“能看到、能听到”：**
    1.  **見える・聞こえる** `$\rightarrow$` 自身能力（自然地看见/听见）。
    2.  **見られる・聞ける** `$\rightarrow$` 人工条件（创造条件后可以看见/听见）。
    *   例：映画館へ行くなら、面白い映画が見られる。（去电影院的话，能看到有趣的电影。）

## 五、做题技巧总结

1.  **翻译** `$\rightarrow$` 确定意思为“使被能”四个当中的谁。
2.  **注意逆向思维：**
    *   ①出现“谁的...” `$\rightarrow$` 优先考虑 **物主被动句**。
    *   ②翻译不太顺 `$\rightarrow$` 考虑 **间接被动句**。
3.  **观察选项：** 根据各自的特点优先选出，但要注意坑，特别是长得像的要留意！
4.  **使役态验证：** 通过闭环故事 `$\rightarrow$` 印证对象的助词符不符合。

## 1. 敬语表达 (尊他语)

**题目：**

> 張先生なら、 もう **（　）** よ。
>
> A. 帰れました
> B. お帰りま した
> C. 帰りたが った
> D. 帰られました

*   **正确答案：** **D. 帰られました**
*   **解析：**
    *   本题考查 **“被动形式表示尊敬”** 的用法。
    *   动词的被动形（～れる/～られる）可以用作**尊他语**，表示对动作主体的尊敬。
    *   **A (帰れました)**：可能态，“能回去了”，语意不符。
    *   **B (お帰りました)**：错误形式。尊他语套路是 `お＋动词连用形＋になる`，这里少了个 `に`。
    *   **C (帰りたがった)**：表示“想回去”。直接描述长辈/他人的欲望是不礼貌的。
    *   **D (帰られました)**：`帰る` 的被动形，在此作为**尊他语**，意为“张老师已经回去了”。

## 2. 使役被动 (被迫做某事)

**题目：**

> 私は飲みたくないです。でも **（　）** のです。
>
> A. 飲まされた
> B. 飲まさせた
> C. 飲まさせられた
> D. 飲まられさせた

*   **正确答案：** **A. 飲まされた**
*   **解析：**
    *   本题考查 **使役被动态**，表示“不想做但被迫做”。
    *   这里的动词是 `飲む` (五段动词/I类动词)。
    *   I类动词的使役被动有两种形式：
        1.  完整形：`飲ま` + `せられる` = `飲ませられる`
        2.  **缩略形**（最常用）：`飲ま` + `される` = **`飲まされる`**
    *   **A** 是正确的缩略形式。
    *   **C** `飲まさせられた` 是错误的冗余形式（`さ` 多余了）。只有 `す` 结尾的动词（如 `話す`）才必须用完整形 `話させられる`。

***

## 3. 使役态 (让/导致...)

**题目：**

> あの話をして社長をすっかり **（　）** しまった。
>
> A. 怒られて
> B. 怒らせて
> C. 怒って
> D. 怒らないで

*   **正确答案：** **B. 怒らせて**
*   **解析：**
    *   本题考查 **使役态**。
    *   句意：“说了那件事，结果**把社长惹生气了**（让社长生气了）”。
    *   助词是 `社長を`，提示后面是使役动词。
    *   **A (怒られて)**：被动，“被骂了/被生气了”。助词通常用 `社長に`。
    *   **B (怒らせて)**：`怒る` 的使役形 `怒らせる`，“让...生气”。符合句意。
